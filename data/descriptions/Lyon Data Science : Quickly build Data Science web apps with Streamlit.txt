Vous avez passé un long moment à perfectionner votre modèle de Machine Learning, il est maintenant temps pour les utilisateurs d'interagir avec, via une application web moderne et responsive !\nToutefois nous sommes un peu démunis face à cette question du déploiement web. Il faut extraire le code de notre patchwork de notebooks Jupyter et apprendre à structurer des API REST en Flask pour assurer les prédictions. Il faut se remémorer nos connaissances poussiéreuses en HTML5/CSS3, et découvrir les débats sur les framework Javascript à la mode pour produire une page web interactive...ou alors laisser la main à un développeur web avec le sentiment de ne plus pouvoir faire évoluer seul cette page.\n\nAprès tout ce travail dans l'écosystème Data Science, l'impression d'avoir à s'initier au développement web pour mettre notre modèle dans les mains du public est handicapante.\nEnter Streamlit (https://www.streamlit.io/), un framework qui vous permet d'implémenter rapidement une application web pour votre modèle sans quitter le monde Pythonic.\nJe vous présenterai le mode de fonctionnement et l'API de Streamlit, comment gérer l'interactivité et la mise en cache des données, le tout avec une démonstration live du rendu visuel -et croisons les doigts un déploiement sur Heroku-.\n\nA la fin de ce stream, vous pourrez vous aussi visualiser les fruits de votre projet Data Science sur le web !\n\nOrateur : Fanilo ANDRIANASOLO, Worldline\n\nPré-requis :\n- Une première expérience en Python est un plus\n- Aucune connaissance web requise
